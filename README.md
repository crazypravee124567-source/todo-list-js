Todo List App

A simple and interactive Todo List web app built with plain HTML, CSS, and JavaScript. It lets you add, complete, and remove tasks — all without any frameworks or installation.



 Live Demo

> Open `todo-list.html` directly in your browser — no setup needed!



 Project Structure


todo-list/
├── todo-list.html   # Main file (HTML + CSS + JS combined)
└── README.md        # Project documentation




Features

- ✅ View all tasks with status (Pending / Completed)
- ✅ Add new tasks using input box or pressing **Enter**
- ✅ Mark tasks as **Completed** (strikethrough effect)
- ✅ Remove tasks from the list
- ✅ Empty state message when no tasks remain
- ✅ Clean and responsive UI

---

 JavaScript Concepts Used

| Concept | Used For |
|--------|----------|
| `Array.push()` | Add new task to list |
| `Array.filter()` | Remove a task from list |
| `Array.forEach()` | Loop through tasks & render them |
| `DOM Manipulation` | Display tasks dynamically on page |
| `Event Listeners` | Handle button clicks and Enter key |
| `Functions` | `addTask()`, `completeTask()`, `removeTask()`, `renderTasks()` |
